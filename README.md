# Dr Yousra El Khadri — Site Web

Site web médical de Dr Yousra El Khadri, médecin esthétique à Casablanca.

## Structure

```
dryousra-website/
├── index.html              Landing page (catégories de soins)
├── a-propos.html           Page À propos avec parcours et congrès
├── partenaires.html        Page Partenaires (Galderma, Merz, Norocpharma)
├── sitemap.xml             Sitemap pour Google
├── robots.txt              Configuration crawlers
├── css/
│   └── style.css           Styles partagés
├── js/
│   └── main.js             Navigation, footer, mega menu, FAQ
├── images/                 Photos, logos, favicons
├── soins/
│   ├── injections/         Botox, Filler, Skin Booster, Dissolution
│   ├── stimulation-collagene/  Sculptura, Radiesse, Harmonica
│   ├── soins-visage/       Hydrafacial, Microneedling, Peeling, Exosomes, Baby Lips
│   ├── soins-corps/        Lipolyse, Peeling corps
│   └── laser/              Laser médical, Détatouage
└── besoins/                Pages besoins patient (7 pages)
```

- Sitemap.xml soumis à Google Search Console
- Schema markup Physician + LocalBusiness sur landing
- Schema FAQPage sur chaque page de soin
- Meta titles et descriptions uniques par page
- Maillage interne entre soins et besoins
